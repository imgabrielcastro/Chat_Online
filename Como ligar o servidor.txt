Olá! Para iniciar o servidor juntamente com o código, basta seguir alguns passos simples.

1 -> Abra o terminal na pasta "back-end" do projeto.
2 -> Execute o seguinte comando para iniciar o servidor em modo de desenvolvimento:

npm run dev
